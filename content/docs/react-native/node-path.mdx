---
title: 安卓编译过程中三方包找不到node路径
---

React Native 模块构建的时候可能找不到node路径，有个小技巧是使用命令行打开android studio，这样android studio就能识别到node路径了。

```bash
cd /Your/Android/Project/Path
open -a "Android Studio"
```

问题解决方案：

方式 1：绕过 node 解析（推荐，改代码）

修改文件：build.gradle
位置：apply plugin 之后、react {} 之前
新增内容（整段）：
目的：让 Reanimated 不再执行 node --print require.resolve(...)，避免找不到 node。

方式 2：让 Gradle 环境能找到 node（不改代码）

代码改动：无（现有 build.gradle 不需要改）
操作改动：
以终端启动 Android Studio（继承 nvm 的 PATH），或
在 Android Studio/Gradle 配置里让 PATH 包含 bin
目的：让 commandLine("node", ...) 在 Gradle 进程里能找到 node。
